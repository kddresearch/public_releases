# Knowledge Discovery in Databases (KDD) Lab — Public Releases
**Institution:** Kansas State University  
**Principal Investigator:** Prof. William H. Hsu  
**Canonical Domain:** [kddresearch.org](https://www.kddresearch.org)  

Welcome to the public-facing open-source repository organization for the KDD Lab. This repository serves as our production release hub for validated research codebases, public datasets, and open academic software packages.

## 🔬 Active Research Domains
Our team executes high-leverage research across the following core tracks:
* **Open Information Extraction (Open IE):** Scalable, open-domain relation extraction and knowledge graph construction.
* **Multimodal Machine Learning:** Automated ingestion, diarization, and semantic tracking of rich video, audio, and streaming social media telemetry.
* **Precision Agriculture:** Reinforcement Learning and computer vision optimizations for agricultural engineering and precision crop management.

## 🗂️ Project Repositories & Open Source Index
The migration, deduplication, and cleanup of our 70 legacy repositories into centralized project structures is currently ongoing. Public releases will be organized by their structural prefixes as they clear our Tier 2 Code Audit gating systems.
